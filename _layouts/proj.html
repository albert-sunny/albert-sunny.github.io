Wifi-Enabled IoT-Based Smart Lock Status: Active Current openings: No
The goal is to design an IOT based smart door-lock which can be operated over a Wifi network using an android application. One would need an authentication mechanism to lock or unlock the door. Hardware implementation of the lock will use an Arduino board, DC motor, NODE MCU ESP8266 Wifi module and batteries to provide power to the lock. This project also involves the design of a locking mechanism (with rack, pinion, and a place fixing the servo motor) and realizing it using a 3D printer.

Secure Linux Voice Command Controller Status: Active Current openings: Yes
This project aims to develop a secure voice-based controller for Linux desktops to recognize voice commands for basic functionalities (for example, the opening of applications and basic control of applications) and perform them similar to commercial virtual assistants. This voice-based controller should be secure, that is it should be able to recognize the speaker and run voice command only for a registered speaker.

SahaYatra - A Ride Sharing App Status: Active Current openings: No
Several attempts on developing ride service platforms have been made. Some are paid, while others are open sourced. But all these projects do not have enough flexibility in terms of ride sharing and ride scheduling. The goal of this project is to fill this void by creating a new ride sharing platform.
